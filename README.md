# ERP Dashboard System 🚀

## 🚀 Overview
This is a responsive ERP Dashboard built using React.js. It allows basic management of employees, projects, and inventory with a clean UI.

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- React Router
- Recharts

## ✨ Features
- Login & Signup UI
- Dashboard with charts
- Employee management with search
- Project tracking with status
- Inventory management
- Responsive design (mobile + desktop)

## 🧱 Architecture
- Reusable Layout component (Sidebar + Navbar)
- Pages rendered using children props
- Component-based structure

## 📱 Responsiveness
- Sidebar toggle on mobile
- Grid-based layout for cards
- Scrollable tables

## 🔮 Future Improvements
- Backend integration
- Authentication system
- Database storage
